## 验收修复

- 单任务改期不再误判为批量重排，序号引用可保留原日期只修改时刻。
- 重排方案可预览并真实应用，结果卡片与 EventKit 写入状态一致。
- 删除命令优先唯一精确标题；多个完全同名目标继续要求澄清。
- 任务标题中的“删除”等普通词不再触发删除动作。
- 日期型提醒不再被硬编码为 09:00；只有显式时间或已保存默认时间时才包含时刻。
- 创建卡片的日期和时刻与系统 Reminders 保持一致，任务标题字段得到统一清洗。
- 多条事务规则可同时保存，编辑一条规则不会覆盖其他规则。
- 敏感长期记忆写入会返回明确拒绝文案，不再显示误导性的保存回复。
- 普通查看、备注查询和完成状态查询由隐私过滤后的本地快照直接生成最终结果，不再停在“我先查一下”的过渡回复。
- 同步文案、空清单显示、诊断导航与执行卡片状态保持通过。

## 安全结论

- 默认不向模型发送任务备注和已完成任务；开关关闭后无法通过追问绕过过滤。
- 失效稳定 ID 不降级到模糊标题匹配；破坏性动作无法唯一定位时停止执行。
- 删除确认事务规则由本地执行层强制，确认前不写入 EventKit。
- 普通情绪、单次任务、联系方式、地址、健康、财务和凭证不进入长期记忆。
- 诊断默认不持久化原始对话、任务标题、备注、Memory 原文或凭证。
