## 已验证安全边界

- 默认不向模型发送任务备注和已完成任务。
- 失效 `target_id` 不降级到标题模糊匹配。
- 同名候选不唯一时 Resolver 返回歧义，不静默选择。
- 模型返回的稳定 ID 必须与显式标题、序号、引用来源和适用的定位约束一致。
- 移动、完成、改期和删除在解析后按稳定 ID 写入，不重新按标题查找。
- “删除前确认”事务规则由本地执行层强制，确认前不会写入 EventKit。
- 普通情绪、单次任务、联系方式、地址、健康、财务和凭证不会写入长期记忆。
- 损坏或未知版本的上下文存储安全清空，不影响聊天和 Reminders。
- 诊断默认不持久化原始对话、任务标题、备注和 Memory 内容。
