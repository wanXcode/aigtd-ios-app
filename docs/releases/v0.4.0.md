# AIGTD Reminders v0.4.0

版本：`0.4.0 (17)`  
主题：上下文与记忆  
状态：Release Candidate

## 本版重点

- 每轮请求使用统一的 `AgentContextSnapshot`，包含最近对话、任务、稳定引用、会话摘要、长期偏好、四类 Agent 文档和隐私描述。
- 创建、改期、移动、完成、删除和查看任务后保存稳定 Reminder ID，后续“刚才那条”“第二条”优先按 ID 解析。
- 同名、失效或不唯一目标停止执行，不再静默选择候选任务。
- 超过最近 8 轮后使用确定性的本地会话摘要保留目标、范围、约束和已执行事实。
- 只有明确表达“以后、默认、总是、记住”等长期语义时才保存偏好；敏感信息和普通聊天拒绝写入。
- Agent 页新增“上下文与隐私”，可控制备注、已完成任务、任务数量并清除本地上下文或长期记忆。

## 隐私默认值

- 任务备注：关闭
- 已完成任务：关闭
- 发送任务上限：40
- 会话上下文保留：30 天
- 失效引用清理：7 天
- 诊断默认仅保存长度、哈希、结构和错误类型

## 已知边界

- 本版仍保持单请求、单 Action，不实现多工具循环或一次执行多个不同动作。
- 会话摘要是本地确定性结构化归纳，不保存模型思考过程。
- 通用自然语言确认、撤销和失败恢复属于后续版本。
